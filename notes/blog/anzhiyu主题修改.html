字数统计 ​

要为 AnZhiYu 配上字数统计特性, 你需要如下几个步骤:

打开 hexo 工作目录

bash
npm install hexo-wordcount --save
或者
yarn add hexo-wordcount
1
2
3

修改 主题配置文件:

yml
wordcount:
  enable: true
  post_wordcount: true
  min2read: true
  total_wordcount: true
